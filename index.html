ABOUT THIS PAGE

Se pretende crear una página web completa que combine estructura (HTML), estilo (CSS), interactividad (JavaScript) y datos externos (JSON vía API). Como resultado, se generará una pequeña aplicación web funcional que resulte de la integración de estos lenguajes.

Se construirá la base de la web en HTML con etiquetas semánticas.

  1. Crea un nuevo archivo llamado 'index.html'.
  2. Define la estructura básica del documento HTML5.
  3. Añade secciones: header, nav, main, aside, footer.
  4. Dentro del main, crea secciones con id='inicio', 'about' y 'api'.
